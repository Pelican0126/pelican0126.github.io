# PaneTrans · Landing Site

Static marketing site for [PaneTrans](https://github.com/Pelican0126/panetrans) — a local-AI Chrome translator with continuous region monitoring.

## What lives here

| File | Purpose |
|---|---|
| `index.html` | Hero + features overview |
| `pricing.html` | Pricing tiers + FAQ |
| `terms.html` | Terms of Service |
| `privacy.html` | Privacy Policy (key claim: zero translation data leaves the user's browser) |
| `refund.html` | 14-day refund policy |
| `style.css` | Single-file stylesheet for everything |
| `assets/` | Logo + screenshots |

## Deploy

The site is plain static HTML — drop it on GitHub Pages, Cloudflare Pages, Netlify, or any static host. The default deploy target is GitHub Pages on this repo.

## Updating screenshots

Screenshots are regenerated by `scripts/capture-readme-shots.js` in the main extension repo and copied here.
