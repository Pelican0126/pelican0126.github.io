What PaneTrans actually does

Drag, then watch.

Right-click anywhere → Translate Region → drag a rectangle. A draggable popup shows the original and the translation side-by-side.

It keeps watching.

A MutationObserver tracks the rectangle. Live chat lines, subtitle changes, ticker updates — the popup retranslates automatically. Like sticking a real-time subtitle strip onto any slice of the page.

Local AI, picked per task.

Free tier: OPUS-MT (~160 MB / pair, real-time on WebGPU) and NLLB-200 / M2M-100 for higher quality. Pro tier: LLM-grade gpt-oss and Gemma 3 for the most fluent output.

Why a Chinese student / engineer / anyone working with English content uses this

  • Zero data leaks. Translation models live in your browser cache. No request to a third-party server. Works offline once the model is downloaded.
  • Persistent, not single-shot. Most translators give you one translation and stop. PaneTrans pins a popup that follows the region until you close it.
  • Bilingual YouTube subtitles as a separate, opt-in feature for video learners.
  • Open architecture. Built on Transformers.js + Xenova ONNX models. You can audit the model list in src/utils/languages.js.
